## 本轮关键决策汇总
- 产品目标：帮助新手司机验证停车思路，原型以“可跑通 + 基本正确”为目标。
- 目标角色：新手司机（单人）。
- 触发条件：用户随时启动；手动触发结束判定。
- 输入范围：4类场景 + 方向盘离散控制 + 动力连续控制。
- 输出范围：前端实时状态 + 最终是否入位；不导出文件。
- 成功判定：车身包围盒在车位框内且最终速度为0。
- 风险处理：压线/碰撞仅标记，可继续，最终统一判定。
- 验收标准：4场景闭环跑通，预设用例判定一致率 >= 90%。
