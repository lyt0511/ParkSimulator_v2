# 任务卡 Slice s01 - 最小用户操作环境构建

## Done When
- 行为完成：
  - 可从 `IDLE -> READY -> RUNNING -> DONE` 跑通一轮最小会话。

  - 点击结束后进入 `DONE`，显示占位结果文本。
- 测试完成：
